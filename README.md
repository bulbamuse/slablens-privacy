# Privacy Policy - GitHub Pages

This repository hosts the privacy policy for [Your Extension Name] web extension.

## Setup

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Under "Source", select the branch (usually `main` or `master`)
4. Save the settings
5. Your privacy policy will be available at: `https://gh-bulba.github.io/[repository-name]/`

## Customization

Edit `index.html` and replace the following placeholders:

- `[Your Extension Name]` - Replace with your actual extension name
- `[Date]` - Add the current date
- `[Permission 1]`, `[Permission 2]` - List your extension's required permissions
- `[your-email@example.com]` - Add your contact email
- Add any third-party services if applicable

## Local Testing

Open `index.html` in your browser to preview the privacy policy before publishing.
